## What is Folio?

Most PDF readers treat you like a filing cabinet. Open file. Read. Close.

Folio is different. It's a **desktop reading environment** built around the idea that reading shouldn't be a passive, lonely activity. You have a resident AI companion ‚ÄĒ **Lyra** ‚ÄĒ who reads alongside you. She can see the page you're on, discuss what you're reading, and she actually has opinions. Not the corporate hedging kind. Real ones.

It runs on your machine. Your books stay on your machine. Nothing phones home.

## Features

<br/>

### ‚ú¶ Lyra ‚ÄĒ an AI that reads with you

> *"No corporate AI energy. No 'As an AI...' garbage. Just sharp, direct, and actually useful."*

Lyra is your AI companion. She's not a chatbot bolted onto a PDF viewer. She's built into the reading experience:

- **She can see what you're reading** ‚ÄĒ sends the current page as an image when you ask (works on comics, diagrams, charts, anything visual)
- **Select any text ‚Üí Ask Lyra** ‚ÄĒ highlight a sentence, click, and she's already on it
- **She has a real voice** ‚ÄĒ powered by ElevenLabs neural TTS, not robotic browser speech
- **Fully customizable persona** ‚ÄĒ rename her, rewrite her personality, make her whoever you need

<br/>

### ūüĖ• Runs completely local (or cloud ‚ÄĒ your choice)

| Provider | Models | Private | Cost |
|----------|--------|---------|------|
| **Ollama** | Llama 3.2, Mistral, DeepSeek R1, Phi-3... | ‚úÖ 100% local | Free |
| **OpenAI** | GPT-4o, GPT-4o Mini | ‚ėĀ API | Pay-per-use |
| **Anthropic** | Claude Opus, Sonnet, Haiku | ‚ėĀ API | Pay-per-use |

No internet required if you use Ollama. Your books, your conversations, your data ‚ÄĒ all stay on your machine.

<br/>

### ūüďĖ Reading experience

- **Eye-safe warm themes** ‚ÄĒ amber-toned paper and dark modes, no harsh blue light
- **5 highlight colors** ‚ÄĒ paint text, click to remove, persists per book
- **Remembers where you left off** ‚ÄĒ opens to your exact scroll position, every time
- **Comic/manga mode** ‚ÄĒ lazy-loads pages so it never lags, even on 400-page volumes
- **Focus mode** ‚ÄĒ strips every distraction, just text and Lyra
- **Fonts** ‚ÄĒ Lora, Georgia, system serif, monospace
- **Column width** ‚ÄĒ Narrow / Normal / Wide / Full

<br/>

### ūüĒä Voice ‚ÄĒ ElevenLabs

Lyra can read her responses aloud. Neural voices, not the robotic browser fallback. Two voices built in, switchable in settings.

## AI Setup

### Option A ‚ÄĒ Ollama (Recommended, fully free)

```bash
# 1. Install Ollama
winget install Ollama.Ollama
# or download from https://ollama.com

# 2. Start it
ollama serve

# 3. In Folio ‚Üí Settings ‚Üí Models ‚Üí pick a model ‚Üí click Get
```

**Recommended models by RAM:**

```
4 GB RAM  ‚Üí  Llama 3.2 1B  (fastest)
6 GB RAM  ‚Üí  Llama 3.2 3B  ‚Üź best balance
8 GB RAM  ‚Üí  Mistral 7B
12 GB RAM ‚Üí  Llama 3.1 8B or DeepSeek R1 7B
```

<br/>

### Option B ‚ÄĒ OpenAI / Anthropic

1. Get an API key from [platform.openai.com](https://platform.openai.com) or [console.anthropic.com](https://console.anthropic.com)
2. Open Folio ‚Üí Settings ‚Üí AI ‚Üí paste your key
3. Pick a model

> **Vision bonus:** OpenAI and Anthropic models can see the page you're reading. Lyra gets sent the current page as an image when you ask her about it.

## Build

```bash
# Clone
git clone https://github.com/Sabbirrahman43/folio.git
cd folio

# Install
npm install

# Dev mode
npm run dev       # terminal 1 ‚ÄĒ Vite
npm run electron  # terminal 2 ‚ÄĒ Electron

# Build Windows installer
npm run build
# ‚Üí release/Folio Setup 1.0.0.exe
```

**Requirements:** Node.js 18+, Windows 10/11 x64
